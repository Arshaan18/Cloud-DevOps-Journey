# Linux Summary

## Commands I Mastered
- Navigation: pwd, cd, ls
- Files: touch, cp, mv, rm
- Permissions: chmod, chown
- Users: useradd, usermod, groupadd
- Processes: ps aux, top, kill
- Networking: ping, nslookup, ss, curl
- Scripting: variables, if/else, loops

## Key Lessons
- chmod 600 = most secure (owner only)
- chmod 755 = standard for scripts
- kill before kill -9 (avoid data corruption)
- Troubleshoot: ping → nslookup → ss → curl
