# Day 6: Linux Networking

## Commands Learned
- ip addr show = show network interfaces and IPs
- ping = check if server is alive
- nslookup = DNS lookup (name to IP)
- ss -tulnp = show open ports
- curl -I = HTTP headers

## HTTP Status Codes
- 200 = OK
- 301 = Redirect
- 404 = Not Found
- 500 = Server Error

## Ports to Remember
- 22  = SSH
- 80  = HTTP
- 443 = HTTPS
- 3306 = MySQL
- 5432 = PostgreSQL

## Troubleshooting Order
ping → nslookup → ss -tulnp → curl
