# Day 7: Git & GitHub Mastery

## Commands Learned
- git log --oneline = see commit history
- git checkout -b branch = create new branch
- git merge branch = merge branch to master
- git branch -d branch = delete branch
- git revert commit-id = safely undo commit
- git push origin --delete branch = delete remote branch

## Workflow
create branch → work → commit → push → PR → merge → delete branch

## Key Lessons
- git add = stage (local)
- git commit = save snapshot (local)  
- git push = send to GitHub (remote)
- PR = code review before merging
- Always use revert not reset in production
