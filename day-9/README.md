## Key Concepts
- S3 = unlimited object storage
- Bucket = container for files
- Object = file in bucket
- Bucket Policy = JSON access rules
- Static hosting = free website hosting

## AWS CLI Commands
- aws configure = setup credentials
- aws s3 ls = list buckets
- aws s3 cp file s3://bucket/ = upload file
- aws s3 sync folder s3://bucket/ = sync folder
