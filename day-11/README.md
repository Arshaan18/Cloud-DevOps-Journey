## What I Did
- Installed Docker on WSL2
- Ran hello-world container

- Ran Ubuntu container interactively
- Created custom Dockerfile
- Built custom Docker image
- Ran custom container on port 8081

## Commands Learned
- docker run = run container
- docker ps = list running containers
- docker ps -a = list all containers
- docker images = list images
- docker stop = stop container
- docker rm = remove container
- docker build = build image
- docker run -it = interactive mode
- docker run -d = detached mode
- docker run -p = port mapping

## Dockerfile Instructions
- FROM = base image
- RUN = execute commands
- COPY = copy files
- EXPOSE = open port
- CMD = startup command

## Key Concepts
- Image = blueprint
- Container = running instance
- Docker vs VM = lightweight, shares kernel
